# egg-bag-framework

该框架，面向于企业中的后端程序，代码下载下来后，自行更改mysql、redis配置

## 开始

```bash
$ npm install
```

## 创建软连接

```bash
$ npm link
$ npm link egg-bag-framework // 应用目录
```
